**The Pantry of the Earth - Botanical Apothecary**

Earthy Munchy is a premium e-commerce experience designed with a warm, botanical, and apothecary-inspired aesthetic. The project focuses on showcasing organic, earth-sourced products like wild honey, ancient millets, and premium spices through a luxurious and interactive user interface.

## âœ¨ Features

- **Moodboard-Driven Design**: A curated color palette featuring Sage Green, Terracotta, Cream, and Clay optimized for a natural feel.
- **Single-File Architecture**: The entire core experience is contained within a single `index.html` file for simplicity and performance.
- **Premium Micro-interactions**:
  - Parallax scrolling effects.
  - Hover-triggered 3D transforms for product jars.
  - Shimmer effects on ingredient cards.
  - Intersection Observer-based fade-in reveals.
- **Responsive Layout**: Fully responsive fluid design using modern CSS Grid and Flexbox.
- **No Heavy Dependencies**: Built with vanilla HTML, CSS, and JavaScript. (Vite is included for development convenience).

## ğŸš€ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd earthymunchy
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the local development server:

```bash
npm run dev
```

This will launch the app at `http://localhost:5173` (or similar).

### Building for Production

To build the project for production:

```bash
npm run build
```

*Note: Since the project is currently optimized as a single-file `index.html`, you can also simply deploy `index.html` directly to any static host.*
